📺 Tải từ link .m3u8 (HLS) — cách lấy link trên trang lạ
  1. Mở trang có video → nhấn F12 (DevTools) → tab Network.
  2. Bấm Play video, gõ m3u8 vào ô lọc của tab Network.
  3. Chuột phải request .m3u8 → Copy → Copy link address.
  4. Dán vào ô trên rồi bấm Tải Ngay — hệ thống tự tải & ghép mọi segment thành 1 file MP4.

💡 Ưu tiên file master.m3u8 (playlist tổng) để chọn được chất lượng cao nhất. Link .m3u8 thường có hạn dùng ngắn — copy xong nên tải ngay.
